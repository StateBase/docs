## 2. Your First Persistent Agent

In this example, we'll build a simple chat loop using **Gemini 2.0-Flash**. StateBase will automatically handle context retrieval and state updates to ensure the agent remembers the user across restarts.

```python
import os
from statebase import StateBase
from google import genai
from google.genai import types

# 1. Initialize Clients
sb = StateBase(api_key="sb_live_...")
gemini = genai.Client(api_key="GEMINI_API_KEY")

# 2. Start (or Resume) a Session
# StateBase Resolve ownership automatically via your API Key
session = sb.sessions.create(
    agent_id="support-agent-v1",
    initial_state={"status": "onboarding", "interaction_count": 0}
)

def chat():
    user_input = input("You: ")
    
    # 3. RETRIEVE: Fetch consolidated context (State + Relevant Memories + History)
    context = sb.sessions.get_context(
        session_id=session.id, 
        query=user_input,
        memory_limit=5
    )
    
    # 4. THINK: Feed the context directly into the System Prompt
    response = gemini.models.generate_content(
        model="gemini-2.0-flash",
        config=types.GenerateContentConfig(
            system_instruction=f"Current Agent State: {context['state']}\nRelevant Memories: {context['memories']}"
        ),
        contents=user_input
    )
    
    ai_output = response.text
    print(f"AI: {ai_output}")
    
    # 5. PERSIST: Save the interaction for auditability and memory
    sb.sessions.add_turn(
        session_id=session.id,
        input=user_input,
        output=ai_output
    )

if __name__ == "__main__":
    chat()
```

## 3. Verify on the Dashboard

Head over to the [StateBase Dashboard](https://app.statebase.org/dashboard/sessions) to see your session in action. You'll find:

- **Complete Trace**: Real-time log of every turn.
- **State Diff**: Exactly how the agent's internal state changed.
- **Memory Bank**: Key facts the agent has extracted and saved for the long term.
