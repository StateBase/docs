## 2. Hello World

Let's build a simple chat loop that remembers user details.

<CodeGroup>
```python Python
import os
from statebase import StateBase
from openai import OpenAI

# Initialize
sb = StateBase(api_key=os.getenv("STATEBASE_KEY"))
client = OpenAI(api_key=os.getenv("OPENAI_KEY"))

# Create a session for a user
session = sb.sessions.create(user_id="alex_doe", agent_id="demo_bot")

while True:
    user_input = input("You: ")
    
    # 1. GET CONTEXT: Auto-fetches relevant long-term memories
    context = sb.sessions.get_context(session.id, query=user_input)
    
    # 2. GENERATE: Feed context to LLM
    response = client.chat.completions.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": f"Context: {context}"},
            {"role": "user", "content": user_input}
        ]
    )
    ai_response = response.choices[0].message.content
    print(f"AI: {ai_response}")
    
    # 3. LOG TURN: StateBase saves the interaction and updates memory
    sb.sessions.add_turn(
        session_id=session.id,
        input=user_input,
        output=ai_response
    )
```

```typescript TypeScript
import { StateBase } from '@statebase/client';
import OpenAI from 'openai';

const sb = new StateBase({ apiKey: process.env.SB_KEY });
const openai = new OpenAI({ apiKey: process.env.OPENAI_KEY });

async function main() {
  const session = await sb.sessions.create({ userId: 'alex_doe', agentId: 'demo_bot' });
  
  // 1. GET CONTEXT
  const context = await sb.sessions.getContext({ sessionId: session.id });
  
  // 2. GENERATE
  const completion = await openai.chat.completions.create({
      messages: [{ role: 'system', content: `Context: ${context}` }],
      model: 'gpt-4',
  });
  
  // 3. LOG TURN
  await sb.sessions.addTurn({
      sessionId: session.id,
      input: "User Input...",
      output: completion.choices[0].message.content
  });
}
```
</CodeGroup>

## What just happened?

By adding **3 lines of code**, you gave your agent:
1.  **Infinite Memory**: It will recall "Alex Doe" in a session 2 weeks from now.
2.  **Audit Logs**: Every turn is visible in the StateBase Dashboard.
3.  **Drift Protection**: If the agent hallucinates, you can rollback the session state.
