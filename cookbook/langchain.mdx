---
title: "LangChain Adapter"
description: "Drop-in memory for LangChain agents"
---

StateBase provides a `BaseMemory` implementation compatible with LangChain.

```python
from langchain.chat_models import ChatOpenAI
from langchain.chains import ConversationChain
from statebase.integrations.langchain import StateBaseMemory

llm = ChatOpenAI(temperature=0)
memory = StateBaseMemory(
    session_id="session_123",
    api_key="sb_live_..."
)

conversation = ConversationChain(
    llm=llm, 
    verbose=True, 
    memory=memory
)

conversation.predict(input="Hello!")
```
